## Steps

1. Install all **npm** dependencies by running following command

   ```
   npm install
   ```

2. Start Electron Application.

   - For Production mode

     ```
     npm start prod
     ```

   - For Development mode

     ```
     npm start dev
     ```

### Linting code

```
npm run lint
```

### Formatting code

```
npm run format
```
